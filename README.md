# RxJS Getting Started

A simple project to show the components inside of RxJS.
Every branch has a specific example of a feature.

## Installation

Npm Users

```bash
npm install
```

Yarn Users

```bash
yarn
```

### Build Code

Npm Users

```bash
npm run build
```

Yarn Users

```bash
yarn build
```

### Run Code

Npm Users

```bash
npm run start
```

Yarn Users

```bash
yarn start
```
